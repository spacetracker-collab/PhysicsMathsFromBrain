# 🧠 Brain-GNN: Emergence of Mathematics and Physics via Graph Rewrite Dynamics

## 📌 Overview

This project implements a **Brain-inspired Graph Neural Network (Brain-GNN)** that explores a fundamental hypothesis:

> **Mathematics and physics can emerge from graph rewrite dynamics (thought processes) in a neural system.**

The system models the brain as a **dynamically rewiring graph**, where:
- Nodes = neurons / concepts  
- Edges = relationships  
- Graph rewrites = thought processes  

Through interaction with data and internal stochasticity ("free will"), the system attempts to **discover laws such as:**
- Arithmetic → `1 + 1 = 2`  
- Physics → `F = m a`  

---

## 🧠 Key Idea

> Thought = **Graph Rewrite + Message Passing**

The system:
1. Learns from data (external world)
2. Rewrites its internal graph (plasticity)
3. Stabilizes patterns (invariants)
4. Extracts laws (mathematics & physics)
